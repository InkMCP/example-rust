# Example Rust

A simple Actix-web server example app, ready to deploy.

## Setup

```bash
cargo run
```

The server runs on port 8080.

## Deploy with Ink

Connect the [Ink MCP server](https://docs.ml.ink/quick-start) to your AI agent and prompt:

> Deploy this Rust app with Ink.
